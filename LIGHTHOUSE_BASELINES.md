# Lighthouse Baselines

## Minimum Targets (Production)
- Performance: >= 75
- Accessibility: >= 90
- Best Practices: >= 90
- SEO: >= 85
- PWA: >= 80

## Audit Cadence
- Run on every production release candidate.
- Store historical scores for regression tracking.

## Required Remediation
If any score is below threshold, create issue with owner and due date before broad launch.
