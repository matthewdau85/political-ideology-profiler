# INSIGHTS METHODS

## Comparative metrics
- Economic/social percentile computed against anonymized response distributions.
- Figure similarity uses Euclidean distance in 2D ideology space.

## Suppression policy
- Subgroup outputs (country, age band, top issues) are not displayed when sample size < 100.

## Presentation standard
All insights are descriptive and non-causal; no deterministic claims about voter behavior are made.
